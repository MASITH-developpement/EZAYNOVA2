# Changelog — eazynova_prix

## [1.0.0] — 2025-11-23

-   Première version stable du module.
-   Ajout du champ calculé « Prix moyen externe » sur les articles.
-   Connexion automatique à l’API Datab (si clé API présente).
-   Fallback scraping Hemea si pas de clé API Datab.
-   Affichage du prix moyen dans la fiche article (onglet Ventes).
-   Documentation d’installation et d’utilisation.

## [1.1.0] — à venir

-   Amélioration de la gestion d’erreur et des logs.
-   Mise en cache optionnelle des résultats.
-   Sélecteur de source prioritaire (Datab/Hemea/autre).
-   Tests unitaires et QA.
