# EAZYNOVA - Tunnel de Vente

Module de création de tunnels de vente progressifs pour Odoo 19 (sans prise de rendez-vous).

## Fonctionnalités

### Backend
- **Création de tunnels** : Créez des tunnels de vente avec plusieurs étapes
- **Formulaires personnalisables** : Configurez les champs pour chaque étape
- **Logique conditionnelle** : Affichez des étapes selon les réponses
- **Mapping CRM** : Connectez les champs aux données CRM/Contacts
- **Suivi des conversions** : Statistiques détaillées par tunnel
- **Intégration CRM** : Création automatique de leads et contacts

### Frontend
- **Multi-étapes** : Navigation fluide entre les étapes
- **Barre de progression** : Indicateur visuel de l'avancement
- **Validation en temps réel** : Vérification des champs
- **Design responsive** : Optimisé mobile et desktop
- **Page de remerciement** : Personnalisable

## Types de tunnels supportés

- Génération de leads
- Qualification de prospects
- Enquêtes et questionnaires
- Inscriptions à des événements
- Demandes de devis
- Téléchargement de ressources
- Formulaires de contact avancés

## Installation

1. Copiez le module dans votre dossier `addons`
2. Redémarrez Odoo
3. Installez "EAZYNOVA - Tunnel de Vente"

## Configuration

### 1. Créer un tunnel

1. Allez dans **Tunnels de Vente > Tunnels**
2. Créez un nouveau tunnel
3. Configurez le type et les options

### 2. Ajouter des étapes

1. Dans l'onglet "Étapes", ajoutez vos étapes
2. Pour chaque étape, définissez les champs du formulaire
3. Configurez le mapping CRM si nécessaire

### 3. Configurer les actions

1. Choisissez si vous voulez créer des contacts/leads
2. Configurez les notifications par email
3. Personnalisez la page de remerciement

### 4. Partager le tunnel

- Le lien est disponible sur chaque tunnel
- Format : `https://votre-domaine.com/funnel/[ID]`
